How to run this program?
1. Clone this program from https://github.com/vvlnote/caesarsCipher.git
2. in the local machine, node.js needed to be installed.
3. open the terminal, and go to where this program is located in the local machine
4. type 
		node caesarsCipher.js
	then there is a default encoded string will be decoded
5. type 
		node caesarsCipher.js "SERR CVMMN!"
	then "SERR CVMMN!" will be read in as encoded string, and screen will print out the decoded string
6. only accept one input encoded string every time run this program in the command line
